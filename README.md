## My Process

### Built With

- Semantic HTML5 markup
- CSS custom properties
- Flexbox
- CSS Grid
- Mobile-first workflow

### What I Practiced

- Structuring cards with semantic elements (`main`, `section`, `article`, `header`, `h2`, `p`)
- Building reusable base card styles and then extending them with modifier classes
- Using Flexbox for card header alignment (avatar + name/status block)
- Using CSS Grid for responsive layout and desktop card placement
- Applying design tokens from a style guide (colors, spacing, typography)

### Continued Development

- Improve speed and confidence with advanced Grid placement
- Keep refining typography hierarchy and visual spacing for closer design matching
- Practice accessibility checks (landmarks, readable contrast, semantic structure)
